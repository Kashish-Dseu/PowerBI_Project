# 🏥 Patient Analysis — Power BI Project

A data analytics project built with **Microsoft Power BI** that visualises and analyses patient records to surface healthcare insights. The `.pbix` report is backed by two raw CSV datasets.

---

## 📁 Repository Structure

```
PowerBI_Project/
├── Patient_Analysis.pbix   # Power BI Desktop report file
├── pr.csv                  # Primary patient records dataset
└── tr.csv                  # Treatment / transaction records dataset
```

---

## 📊 Project Overview

| Detail | Info |
|--------|------|
| **Tool** | Microsoft Power BI Desktop |
| **Domain** | Healthcare / Patient Analytics |
| **File type** | `.pbix` (Power BI report) |
| **Data sources** | `pr.csv`, `tr.csv` |

---

## 🔍 Features

- **Patient Records Analysis** — Explore patient demographics, admission details, and key attributes from `pr.csv`.
- **Treatment & Transaction Insights** — Analyse treatment patterns and related data from `tr.csv`.
- **Interactive Dashboards** — Slice and filter data visually using Power BI's built-in interactivity.
- **Data Visualisations** — Charts, KPIs, and tables built to communicate trends clearly.

---

## 📂 Data Files

| File | Description |
|------|-------------|
| `pr.csv` | Patient records — contains patient demographics and stay details |
| `tr.csv` | Treatment records — contains treatment/transaction-level data |

---

## 🛠️ Technologies Used

- Microsoft Power BI Desktop
- CSV (data source)
- DAX (Data Analysis Expressions) for measures and calculations
- Power Query (M language) for data transformation
